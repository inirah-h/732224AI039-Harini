- Sign in to n8n.
- Create a new workflow.
- Add a "Schedule Trigger" node.
- Set the schedule time (e.g., every day at 9:00 AM).
- Add another node and search for “Gmail”.
- Select “Gmail: Send Email”.
- Enter the recipient email address (To).
- Type the subject and message body.
- Connect the Schedule Trigger → Gmail Send Email nodes.
- Save and activate the workflow.
